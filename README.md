# Simple Button Role System

A lightweight JavaScript-based **click-to-role button system** designed for simplicity and fast integration.

## Features
- Minimal and fast  
- Easy to integrate  
- Modular structure  
- No unnecessary dependencies  

## Usage
Simply include the script and attach your logic to button events.

```javascript
button.addEventListener("click", () => {
  // your role logic here
});
